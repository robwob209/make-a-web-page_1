Lesson 1: The Basics of the Web and HTML

Web Pages

A web page is text document written in a language called HTML, or HyperText Markup Language. People use browser to request data from servers via the Internet. The servers return the information back to the browser which interprets the data.

Coding

Coding happens when programmers write tex in a language that a computer can understand. We learned some simple coding using HTML, Tags, Elements, and Attributes.
Tag : A tag is always contained within angled brackets. Most tags have an opening tag (for example < b > and a closing tag < /b >).Some tags do not require a closing tag (like the < br >, line break tag). They are called "void'' tags.
Element: An element refers to everything within a set of opening and closing tags.
Attribute: An attribute is a property of an element, used to set the href attribute on an anchor tag. Used for making links, URL resources.

Computers are stupid

Computers need to be told exactly what to do, using the correct "syntax". If you forget to close a < b > tag, the computer will not be able to figure out what you had intended to make bold. It could be very frustrating and sometimes time consuming. Did I mention computers are stupid?

Lesson 2: Creating a Structured Document with HTML

Developer Tools

Developer Tools (in the Browser, like Google Chrome) have powerful feauture that will allow you to see the DOM structure of a document and it allows you to dig through the embedded block elements/boxes.

Text Editors (for programming)

Scratchpad, Codepen, and Sublime Text, are editors that programmer's use. They have many powerful features that make programming easier. For example, some text editors will automatically generate a closing HTML tag when you write an opening tag and will let you see in real time the changes you've made.

HTML Elements

HTML elements are either inline or block. Block elements create an "invisible box" around the content inside of them.

DOM

DOM (Document Object Model), the "tree-like structure" of an HTML page. It defines the logical structure of documents and the way a document is accessed and manipulated. In a browser, this structure shows up as a series of nested boxes. There are boxes inside of boxes inside of boxes, and so on...
